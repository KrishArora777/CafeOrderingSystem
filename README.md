# Python Cafe Billing System ☕

This is a beginner-friendly Python project created to practice fundamental Python concepts.

## Features
- Welcome message
- User input (Name & Age)
- String manipulation (`strip()`, `title()`, `split()`)
- Menu item validation using `in`
- Bill calculation using arithmetic operators
- Discount system using `if-elif-else`
- VIP customer check using nested `if`
- Receipt design using nested `for` loops

## Concepts Used
- Variables
- Input/Output
- Strings
- Operators
- Conditional Statements
- Nested If-Else
- Nested Loops
- Membership Operators

## Language
- Python 3
